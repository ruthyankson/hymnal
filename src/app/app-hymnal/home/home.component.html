Welcome! This app provides hymn lyrics from the Only Believe Ghanaian Hymnal.

Search Tips:

  • By Title: Type any word from the hymn title.
  • By Stanza or Chorus: Check the corresponding box and type any word you recall.
  • All: Check the "All" box to search by title, stanza, and chorus.

Note: Maximum number of hymns available is 400.
